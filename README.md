# Agency CRM System

Профессиональная CRM система для управления клиентами, проектами и финансами рекламного агентства.

## Функционал

- Управление клиентами и контактами
- Проекты и задачи
- Финансовый учет
- Документооборот
- Аналитика и отчеты

## Технологический стек

- **Frontend**: Next.js 15 + TypeScript
- **Стилизация**: CSS Modules + SCSS
- **Шрифт**: Inter

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.
